今天唐大牛教了些git的命令。顺便把他的git别名也发给我了。啥叫别名，就是
{% blockquote %}
git br
==>
git branch
{% endblockquote %}
只需要在home目录下的.gitconfig文件中添加：
```
[alias]
  st = status -s
  ci = commit
  l = log --oneline --decorate -13
  ll = log --oneline --decorate
  co = checkout
  br = branch
  rb = rebase
  dci = dcommit
```

以后就可以使用别名了。
